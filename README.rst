Overview
========
This is a simple CLI English-Japanese dictionary on `goo辞書 <http://dictionary.goo.ne.jp/ej/>`_

You can look up an English word like this.

    $ goodict fujisan
    $ goodict monetary policy
    $ goodict day after day

REQUIREMENTS
============
* mechanize
* hpricot
